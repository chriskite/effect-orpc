# Security Policy

## Reporting a Vulnerability

If you discover a security issue in `effect-orpc`, please report it privately
via GitHub's [private vulnerability reporting](https://github.com/utopyin/effect-orpc/security/advisories/new).

Please do **not** open a public issue for security reports. We aim to triage
incoming reports within seven days.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x     | ✅        |

Pre-1.0 versions only receive fixes on the latest minor.
